### Use CDN

```html
<script src="https://unpkg.com/smartblock@1.3.1/dist/smartblock.js"></script>
```

### Install From npm

via npm

```sh
$ npm install smartblock --save
```

or yarn

```sh
$ yarn add smartblock
```
